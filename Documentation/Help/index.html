HTML structure - top

This template build using Bootstrap framework version 3. All the information within main content is nested within a div with id- name "content". If you need more information you can read Bootstrap documentation on this link

 

1. Sample html structure for content
<div id="content">
	<div class="container">
		<div class="row">
			<div class="col-md-4">
				this is your content			
			</div>
			<div class="col-md-8">
				this is your content
			</div>
		</div>
	</div>
</div>

 

2. You can create column from 12 predefined grid size, then insert your content inside.
<div class="row">
  	<div class="col-md-1">your content</div>
	<div class="col-md-2">your content</div>
	<div class="col-md-3">your content</div>
	<div class="col-md-4">your content</div>
	<div class="col-md-5">your content</div>
	<div class="col-md-6">your content</div>
	<div class="col-md-7">your content</div>

CSS Files - top

These are css files used in this template:
  1. style.css - main css files (you can edit color, font, etc from this css)
  2. bootstrap.css - css framwork for this template
  3. prettyPhoto.css - used for image popup
  4. owl.carousel.css - used for carousel
  5. supersized.css - used for background slider
  6. main.css - used for load all css files

You can change almost all styles from style.css file.

Setting Contact Page- top

Open > Send.php. Change $email_to value with your email..

Setting Countdown Timer - top

  • Open file js/countdown-custom.js
  • Look code below 
    $('#defaultCountdown').countdown({until: new Date(2017, 4-1, 10, 8)}); // year, month, date, hour
  • Change Date value above with any date. The format is year, month, date, hour.
